What is the name of your app?
FixYourStuff.

What collaborative activity can people do together using your app?
Users can help one another to fix their stuff.


What is the ideal number of concurrent users (must be at least 10 but no more than 40)?
The ideal number is between 30 to 40.

What skills do people need in order to do this activity?
People need to be proficient in the category they enrolled in.

Is this activity for fun or for something practical?
It is for practical use and it can be for fun.

How can a person see what everyone else is doing?
Each user has its own profile. The profile has some information about which item the user is responsible for to fix

How can a person see the most recent result of everyone's collaborative effort?
As soon as the issue is fixed, the user has to send a broadcast for the solution.

What can an admin see (i.e., God's mode)?
Admins can see a list of all of the current users and issues.
